### Тема #6. «Тестирование функций с pytest».

Инструкция по запуску тестов:

# Запуск всех тестов
python -m unittest discover

# Запуск конкретного модуля
python -m unittest test_python.py
python -m unittest test_filemanager.py
python -m unittest test_filemanager_mock.py

# Запуск с подробным выводом
python -m unittest -v test_python.py

# Запуск конкретного теста
python -m unittest test_python.TestBuiltinFunctions.test_filter_even_numbers

Что тестируется:

# В test_python.py:

    filter - 7 тестов (четные/нечетные числа, строки, None значения и т.д.)

    map - 7 тестов (квадраты чисел, преобразование строк, несколько списков и т.д.)

    sorted - 10 тестов (числа, строки, сложные объекты, пустые списки и т.д.)

    math.pi - 4 теста

    math.sqrt - 6 тестов

    math.pow - 6 тестов

    math.hypot - 8 тестов

    Комбинации функций - 4 теста

# В test_filemanager.py:

    Банковский счет - 10 тестов (загрузка/сохранение данных, бизнес-логика)

    Викторина - 8 тестов (валидация вопросов, подсчет очков, проверка ответов)

    Пути - 5 тестов

    Вспомогательные функции - 5 тестов

# В test_filemanager_mock.py:

    Создание папки - 3 теста с моками

    Удаление - 3 теста с моками

    Копирование - 3 теста с моками

    Просмотр содержимого - 2 теста

    Смена директории - 3 теста

    Банковский счет с моками - 3 теста
